📘 About This App

This web application predicts the estimated insurance cost for an individual based on various health and demographic factors.

The model is trained using a dataset containing age, gender, BMI, number of children, smoking status, and region.

  • 📊 Machine Learning Model: RandomForestRegressor
  • 🔍 Predicts insurance charges accurately from user inputs
  • 🧠 Built using Python, Pandas, Scikit-learn, Flask

This project is ideal for learning ML deployment and understanding how health metrics affect insurance pricing.
